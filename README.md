# Muhammad Hasib's Portfolio Website

## Description

This repository contains the source code for Muhammad Hasib's personal portfolio website. The website showcases Muhammad's skills, projects, and blog posts related to web design and development, with a focus on Python and JavaScript.

## Live Demo

You can view the live website at: [https://muhamadhasib.github.io/portfolio](https://muhamadhasib.github.io/portfolio)

## Features

- Responsive design for optimal viewing on various devices
- Dark/Light mode toggle for user preference
- Sections for Home, Resume, Portfolio, Blog, and Contact
- Interactive elements like skill bars and animated circles
- Blog section with multiple articles on programming topics
- Contact form for easy communication
- Social media links and professional information

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome for icons
- Prism.js for code syntax highlighting in blog posts

## Structure

- `index.html`: Main page of the website
- `blog.html`: Blog listing page
- `blog1.html`, `blog2.html`, `blog3.html`: Individual blog post pages
- `styles.css`: Main stylesheet
- `script.js`: JavaScript file for interactive features
- `images/`: Directory containing all images used in the website

## Setup for Local Development

1. Clone the repository:
   ```
   git clone https://github.com/muhamadhasib/portfolio.git
   ```
2. Open `index.html` in your preferred web browser.
